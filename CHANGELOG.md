# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-14

### Added
- Initial release of UnityDebugSheet Hierarchy Extension
- Scene hierarchy viewer for UnityDebugSheet
- GameObject search functionality
- Component inspector pages for common Unity components:
  - Transform / RectTransform
  - Camera, Light
  - MeshRenderer, SpriteRenderer
  - Rigidbody / Rigidbody2D
  - Animator, Animation
  - ParticleSystem
  - AudioSource, AudioListener
  - Canvas, CanvasGroup, CanvasScaler
  - Image, RawImage, Button, Slider
  - TextMeshProUGUI
  - BoxCollider
- Unity Package Manager support via Git URL
